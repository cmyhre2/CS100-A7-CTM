C# Overview

History

C# is an object oriented programming language created in 2002 by a team of programmers lead by Anders Hejlsberg at Microsoft. The language was developed within its .NET framework and later approved to become a standard by ECMA.

Common Use Cases

  • Game Development
  • Cloud Based Services
  • Virtual Reality

Current State

C# is still one of the most popular programming languages mainly used in Microsoft's .NET framework and in areas such as game development and cloud based services.
