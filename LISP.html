LISP Overview

History

LISP is an object-oriented language that was created in 1960 by John McCarthy. LISP was a pioneer of its time being one the very first programming languages. Over time, LISP would evolve to become the "common" language of AI.

Common Use Cases

  • Machine Learning and AI
  • Graphics
  • Data Analysis

Current State

Currently LISP as its original form is not used rather it has evolved into several forks of the original language such as LISP Common which is used in AI nowadays.
