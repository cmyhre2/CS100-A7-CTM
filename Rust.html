Rust Overview

History

Rust is a programming language created in 2006 by Graydon Hoare. Rust started out as one of Hoare's personal projects but grew into something big being maintained by the Rust Group nowadays. The language was first announced in 2010 and a stable v1.0 release that came in 2015. The language gained popularity with it's ephasis on safety and performance. The Rust Group is backed by big corporations like Google, Amazon and Mozilla.

Common Use Cases

  1. System-level Programming
  2. Web Development
  3. Data Management

Current State

Rust remains one of the most popular programming languages to this day. Rust is still used for memory management, memory safety and better performance.
