PHP Overview

History

PHP was oringinally a set of web tools created in 1994 by Rasmus Lerdorf. (left picture below) PHP formally known as "PHP/FI" was a product that implemented a set of CGI binaries written in C. Version 3.0 PHP/FI would be renamed to simply, "PHP". As time passed, Lerdorf would add more tools to PHP making its usefullness towards web development and server managment compound.

Common Use Cases

  • Server-side Web Development
  • Form Handling
  • Database Interaction

Current State

PHP is still widely used, especillay in web development platforms like WordPress, Wix and Blogspot.
